# Student Performance Prediction Using Machine Learning

## Project Overview
This project predicts student pass/fail performance using machine learning techniques based on student study habits, attendance and lifestyle factors.

## Dataset Information
- Dataset: Student Performance Factors
- Total Records: 6607
- Total Features: 20
- Target Variable: Pass_Fail

## Machine Learning Workflow
1. Data Cleaning
2. Missing Value Handling
3. Exploratory Data Analysis (EDA)
4. Target Variable Creation
5. Feature Encoding
6. Data Splitting
7. Model Training
8. Evaluation

## Project Structure
- `dataset/` → dataset files
- `src/` → Python source code
- `results/` → EDA charts and outputs
- `notebooks/` → notebooks and experiments

## Models Used
- Logistic Regression
- Decision Tree
- Random Forest

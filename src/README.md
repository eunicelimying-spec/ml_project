# Student Performance Prediction Project

This project predicts whether a student will pass or fail based on student performance factors.

## Dataset

The dataset used is included in this repository:

StudentPerformanceFactors.csv

## Project Files

- milestone1.py: Loads dataset, cleans missing values, creates target variable, performs EDA, encoding, and data splitting.
- milestone2.py: Builds the architecture logic using Logistic Regression and Random Forest.
- milestone3.py: Runs the training loop using Random Forest with different numbers of trees.
- milestone4.py: Performs model optimization using GridSearchCV.
- milestone5.py: Performs final testing, confusion matrix, ROC curve, error analysis, and feature importance.
